# GenerateHousieTicket


~~~
Write a Java program which would randomly generate a Housie Ticket, Below are the rules generated for ticket.

1. There are 3 rows and 9 columns in the ticket. All column values should be between 1-90. 
    In the first column the values of the numbers are 1-10, second column numbers are 11-20 ...
2. Each column has max 2 cells filled in.
3. The numbers can't be repeated in column.
4. Each row has max 5 cells fille, hence only 15 cells in all.

~~~
